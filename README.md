Pharmacy Intelligence Engine v1.0
A Data Science Solution for Healthcare Logistics

Context
This project bridges the gap between clinical microbiology and data analytics. Developed as a core asset for Vision 2036, it addresses the critical challenge of pharmaceutical stock-outs in regional settings.

Key Features
- SQL Data Foundation: Structured relational database for pharmaceutical inventory.
- Predictive Restock Analysis: Automated identification of stock-outs.
- Capital Optimization: Calculates exact financial investment needed for restocking.

Deployment
1. Run `sql/schema.sql` in your MySQL environment.
2. Install dependencies: `pip install pandas sqlalchemy pymysql`.
3. Execute `src/main.py` to generate the Procurement Report.
